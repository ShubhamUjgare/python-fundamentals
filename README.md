## 📖 What Each Day Covers

### Day 1 — OOP Basics (`day1_oop/`)
- Class and object creation
- `__init__` constructor
- `self` keyword
- Instance variables vs class variables
- Methods inside a class
- `__str__` method
- Inheritance — parent and child class
- `super()` keyword
- Method overriding

### Day 2 — Lists Deep Dive (`day2_lists/`)
- Creating lists and indexing
- Positive and negative indexing
- Slicing — `arr[start:end:step]`
- Reverse using `arr[::-1]`
- Adding — `append()`, `insert()`, `extend()`
- Removing — `pop()`, `remove()`, `del`
- Searching — `in`, `index()`, `count()`
- Sorting — `sort()`, `sorted()`, `sort(key=lambda)`
- List comprehensions
- `enumerate()` and `zip()`
- 2D lists and matrix operations
- List as Stack pattern

### Day 3 — Dictionaries (`day3_dicts/`) ⬜
- Creating and accessing dicts
- `get()`, `in` check, `items()`, `keys()`, `values()`
- Nested dictionaries
- `defaultdict` from collections
- Dict comprehensions
- HashMap pattern for DSA

### Day 4 — Sets (`day4_sets/`) ⬜
- Creating sets
- `add()`, `remove()`, `discard()`
- `in` check — O(1) vs list O(n)
- Union, intersection, difference
- Set comprehensions
- HashSet pattern for DSA

### Day 5 — Strings + Tuples (`day5_strings_tuples/`) ⬜
- String slicing and indexing
- `split()`, `join()`, `strip()`
- `upper()`, `lower()`, `replace()`
- `ord()` and `chr()` for char problems
- String reverse using slicing
- f-strings
- Tuple creation and unpacking
- Immutability concept

### Day 6 — Counter + defaultdict + deque + heapq (`day6_counter_deque_heapq/`) ⬜
- `Counter` — frequency counting in one line
- `defaultdict` — no KeyError ever
- `deque` — O(1) append and popleft for BFS
- `heapq` — min-heap push/pop
- Max-heap using negation trick

### Day 7 — Big O + Consolidation (`day7_bigo/`) ⬜
- O(1) — constant time
- O(n) — linear time
- O(n²) — quadratic time
- O(log n) — logarithmic time
- O(n log n) — sorting
- Compare list vs dict vs set lookup
- Push all week code to GitHub
